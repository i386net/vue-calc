# vue-calculator

Простейший калькулятор, стилизованный под калькулятор на macOS.

Node, Vue, Babel, BEM

![calc](./calc.png)

## Настройка
```
npm install
```

### Сборка для разработки с hot-reload
```
npm run serve
```

### Сборка для продакшена
```
npm run build
```

### Линтинг
```
npm run lint
```
